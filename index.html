How to launch this website for free

Follow these steps to put Rosy's Kitchen live on the internet using GitHub Pages — completely free, no hosting fees ever.

1
Create a GitHub account
Go to github.com and sign up for a free account if you don't have one. If you already have one, just sign in. You only need the free plan.
2
Create a new repository
Once signed in, click the + icon at the top right → New repository. Name it exactly: rosys-kitchen — tick "Add a README file" — click Create repository.
3
Upload the website file
Inside your new repository, click Add file → Upload files. Drag and drop the index.html file you downloaded. Scroll down and click Commit changes.
Make sure the file is named exactly index.html — not index.html.txt or anything else.
4
Turn on GitHub Pages
In your repository, click Settings (top menu) → scroll down to Pages in the left sidebar. Under "Source", select Deploy from a branch. Set branch to main and folder to / (root). Click Save.
5
Your site is live!
Wait 2–3 minutes, then visit your website at:

https://YOUR-GITHUB-USERNAME.github.io/rosys-kitchen

Replace YOUR-GITHUB-USERNAME with your actual GitHub username. Share this link with customers!
Example: if your username is kwame123, the link is https://kwame123.github.io/rosys-kitchen
6
How to update the site later (add photos, change prices)
When you have food photos or want to update prices: go to your repository on github.com → click index.html → click the pencil icon to edit → make your changes → click Commit changes. The live site updates automatically within 1–2 minutes.
7
Optional: get a custom domain (e.g. rosyskitchen.com)
Buy a domain from Namecheap or GoDaddy (~$10/year). In GitHub Pages settings, add your custom domain in the "Custom domain" field. Then in your domain registrar, add a CNAME record pointing to your GitHub Pages URL. Full instructions at: docs.github.com/pages/configuring-a-custom-domain
